## 一、概述

本文档描述五通道自动测试系统的测试指标规格，以及测试数据导出到Excel模板时的索引命名规则。测试人员可依据本文档设计和填写Excel报告模板，系统将自动根据索引将测试数据填充到对应位置。

本系统支持的测试项包括：动态范围测试、通道增益测试、带外抑制测试、带内增益波动测试、镜像抑制测试、谐波抑制测试、邻道抑制测试、端口驻波测试。所有测试均在3330MHz至3390MHz范围内的四个频点进行。

---

## 二、测试项列表

### 2.1 动态范围及中频输出信号测试

| 属性 | 值 |
|------|------|
| 测试项名称 | 动态范围及中频输出信号测试 |
| 索引前缀 | dt |
| 测试描述 | 验证接收通道在指定输入功率范围内的输出响应及中频输出信号幅度 |
| 测试单位 | dBm |
| 测试频率 | 3330MHz、3350MHz、3370MHz、3390MHz |
| 测试带宽 | 10kHz |

**测试说明**：本测试在不同输入功率条件下（-90dBm、-50dBm、-10dBm）测量接收机的中频输出信号强度，用于评估接收机的动态范围和线性度。测试时需设置信号源输出指定频率和功率的信号，接收机工作在指定衰减条件下，频谱仪读取中频输出信号的功率值。

### 2.2 通道增益测试

| 属性 | 值 |
|------|------|
| 测试项名称 | 通道增益测试 |
| 索引前缀 | zy |
| 测试描述 | 测量接收通道的固定增益及衰减步进精度 |
| 测试单位 | dB |
| 测试频率 | 3330MHz、3350MHz、3370MHz、3390MHz |
| 测试带宽 | 10kHz |

**测试说明**：本测试测量接收通道在各频点的增益值。测试时信号源输出指定频率的信号，接收机工作在零衰减状态，测量并计算通道的实际增益值。通过比较不同频点的增益差异，可评估通道的频率响应特性。

### 2.3 带外抑制测试

| 属性 | 值 |
|------|------|
| 测试项名称 | 带外抑制测试 |
| 索引前缀 | dw1、dw2 |
| 测试描述 | 测量通道对工作频率带外信号的抑制能力 |
| 测试单位 | dB |
| 测试频率 | 3330MHz、3350MHz、3370MHz、3390MHz |
| 测试带宽 | 70kHz |

**测试说明**：本测试评估接收机对带外干扰信号的抑制能力。测试时信号源输出带外频率的信号（通常为工作频率的整数倍偏移），测量接收机对该信号的抑制程度。索引dw1和dw2对应相同的测试数据，用于满足报告中不同位置的填写需求。

### 2.4 带内增益波动测试

| 属性 | 值 |
|------|------|
| 测试项名称 | 带内增益波动测试 |
| 索引前缀 | dn |
| 测试描述 | 测量通道在5MHz带宽内的增益平坦度 |
| 测试单位 | dB |
| 测试频率 | 3330MHz、3350MHz、3370MHz、3390MHz |
| 测试带宽 | 5kHz |

**测试说明**：本测试测量接收通道在指定带宽内的增益变化幅度。通过在中心频率附近多个频点进行测量，计算最大增益与最小增益的差值，评估通道的带内平坦度性能。

### 2.5 镜像抑制测试

| 属性 | 值 |
|------|------|
| 测试项名称 | 镜像抑制测试 |
| 索引前缀 | jx |
| 测试描述 | 测量通道对镜像频率信号的抑制能力 |
| 测试单位 | dB |
| 测试频率 | 3330MHz、3350MHz、3370MHz、3390MHz |
| 测试带宽 | 10kHz |

**测试说明**：本测试评估接收机对镜像频率信号的抑制能力。镜像频率通常是与本振频率对称于信号频率的另一频率点。测试时信号源输出镜像频率的信号，测量接收机对该信号的抑制程度。

### 2.6 谐波抑制测试

| 属性 | 值 |
|------|------|
| 测试项名称 | 谐波抑制测试 |
| 索引前缀 | xb |
| 测试描述 | 测量中频输出信号的谐波抑制水平 |
| 测试单位 | dBc |
| 测试频率 | 3330MHz、3350MHz、3370MHz、3390MHz |
| 测试带宽 | 50kHz |

**测试说明**：本测试测量接收机输出信号中谐波分量的抑制水平。测试时信号源输出指定频率信号，频谱仪测量中频输出信号及其二次、三次谐波的功率差值，评估谐波抑制性能。

### 2.7 邻道抑制测试

| 属性 | 值 |
|------|------|
| 测试项名称 | 邻道抑制测试 |
| 索引前缀 | ld |
| 测试描述 | 测量通道对相邻信道干扰信号的抑制能力 |
| 测试单位 | dB |
| 测试频率 | 3330MHz、3350MHz、3370MHz、3390MHz |
| 测试带宽 | 50kHz |

**测试说明**：本测试评估接收机对相邻信道干扰信号的抑制能力。测试时在信号源输出工作频率信号的同时，在偏移15MHz处输出另一路干扰信号，测量接收机对干扰信号的抑制程度。**注意**：本测试项的最终结果需在测量值基础上加20dB后再记录。

### 2.8 端口驻波测试（输入）

| 属性 | 值 |
|------|------|
| 测试项名称 | 端口驻波测试（输入） |
| 索引前缀 | zb |
| 测试描述 | 测量接收机输入端口的电压驻波比 |
| 测试单位 | 无量纲 |
| 测试频率 | 3330MHz、3350MHz、3370MHz、3390MHz |
| 测试带宽 | N/A |

**测试说明**：本测试使用矢量网络分析仪（ZNB8）测量接收机输入端口的电压驻波比（VSWR）。测试前需先完成网络分析仪的校准，并将测试线缆连接至接收机输入端口。测试时设置ZNB8扫描频率范围为3300MHz至3400MHz，在各测量频点读取反射系数S11，计算得到驻波比值。

## 三、索引命名规则

### 3.1 索引格式定义

Excel导出索引采用以下格式，索引需用方括号包围：

```
[前缀][通道字母][频率序号][功率后缀]
```

**各部分说明**：

| 位置 | 含义 | 取值范围 |
|------|------|----------|
| 第1-2位 | 测试项前缀 | dt、zy、dn、dw1、dw2、jx、xb、ld、zb |
| 第3位 | 通道标识 | a、b、c、d、e（对应通道1-5） |
| 第4-5位 | 频率序号 | 30、50、70、90（对应3330、3350、3370、3390MHz） |
| 第6-8位 | 功率后缀 | 仅动态范围测试使用：-90、-50、-10 |

### 3.2 索引示例

| 索引示例 | 含义 |
|----------|------|
| [dta30-90] | 动态范围测试，通道1，3330MHz，输入功率-90dBm |
| [dtb50-90] | 动态范围测试，通道2，3350MHz，输入功率-90dBm |
| [zba70] | 端口驻波测试（输入），通道3，3370MHz |
| [zya30] | 通道增益测试，通道1，3330MHz |
| [dw1c50] | 带外抑制测试，通道3，3350MHz |
| [ldd90] | 邻道抑制测试，通道4，3390MHz |

## 七、Excel模板制作说明

### 7.1 模板文件位置

将Excel模板文件命名为`测试报告模板.xlsx`，放置在程序运行目录下的`Template`文件夹中：

```
[程序目录]
├── Template/
│   └── 测试报告模板.xlsx
└── 五通道自动测试.exe
```

### 7.2 填写索引

在Excel模板中，需要填写测试数据的单元格内输入对应的索引标识（用方括号包围）。系统启动时会自动扫描模板中的所有索引，并建立索引到单元格的映射关系。

**示例**：

| A | B | C |
|---|---|---|
| 动态范围 | 通道1-3330MHz | [dta30-90] |
| | 通道1-3350MHz | [dta50-90] |
| | 通道2-3330MHz | [dtb30-90] |

### 7.3 导出流程

1. 运行测试程序，完成各通道的测试
2. 点击"导出报告"按钮
3. 选择保存位置和文件名
4. 系统自动将测试数据填充到模板中对应的索引位置
5. 导出完成，提示用户

## 八、仪表配置

### 8.1 所需仪表

| 仪表名称 | 型号/规格 | 用途 |
|----------|-----------|------|
| 频谱仪 | Keysight N9020A | 测量信号功率、谐波、邻道抑制等 |
| 信号源 | Keysight E4438C | 提供测试激励信号 |
| 矢量网络分析仪 | ZNB8 | 测量端口驻波比 |
| 工装板 | 自研 | 通道切换和频点控制 |

### 8.2 连接方式

- 频谱仪：通过USB或GPIB连接
- 信号源：通过USB或GPIB连接
- 矢量网络分析仪：通过TCP/IP连接（默认地址：TCPIP0::192.168.0.252::inst0::INSTR）
- 工装板：通过串口连接
